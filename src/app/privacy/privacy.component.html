Política de privacidad

En Treasure Summit respetamos la privacidad de los usuarios. Esta página explica de forma sencilla qué información puede recopilarse al utilizar esta web.

1. Responsable del sitio

Esta web ha sido desarrollada como proyecto personal. Puedes contactar conmigo a través de la sección de contacto del sitio web.

2. Datos que puede guardar la web

Treasure Summit puede guardar el nombre del jugador, la puntuación obtenida, el número de preguntas acertadas y el progreso alcanzado en el ranking del juego.

Estos datos se utilizan únicamente para mostrar el ranking público de puntuaciones dentro de la web.

3. Uso de servicios externos

Esta web puede utilizar servicios externos como Supabase para guardar los resultados del ranking y Google AdSense para mostrar anuncios.

Google AdSense puede utilizar cookies o tecnologías similares para mostrar anuncios y medir su rendimiento.

4. Cookies

La web puede utilizar cookies propias o de terceros necesarias para el funcionamiento del sitio, análisis o publicidad.

El usuario puede configurar o bloquear las cookies desde las opciones de su navegador.

5. Finalidad de los datos

Los datos del ranking se utilizan para mostrar las puntuaciones de los jugadores. Los datos relacionados con publicidad se utilizan para mostrar anuncios y mejorar su funcionamiento.

6. Contacto

Para cualquier duda relacionada con esta política de privacidad, puedes utilizar la sección de contacto de la web.
